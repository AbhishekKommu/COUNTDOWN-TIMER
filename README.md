⏳ Beautiful Countdown Timer

A modern and responsive Countdown Timer built using HTML, CSS, and JavaScript. Users can enter the desired time in minutes, start the countdown, and reset it whenever needed. The project features an attractive UI with a beautiful gradient background.

🚀 Features
⏳ Countdown timer in minutes and seconds
▶️ Start button to begin countdown
🔄 Reset button to restart timer
🎨 Beautiful gradient background
📱 Responsive design
⚡ Fast and lightweight
💻 Beginner-friendly project

▶️ How to Run
Download or clone the repository.
Open timer.html in your browser.
Enter the time in minutes.
Click Start to begin the countdown.
Click Reset to reset the timer.
